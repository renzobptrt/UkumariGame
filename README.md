# UkumariGame 
Vertical platform Rogue-like has as main character a ancient ruins explorer. 
Compiled in Unity 2019.4.11f version

## Features
- Type: 2D
- Point of view: Third person
- Gender: Platform - Rogue - like
- Platform: Mobile
- Graphics engine: Unity 2D
- Main language: C#
